# Denofy

*Work in progress* 

Hi! Welcome to this little proyect called Denofy, in reference to Spotify. Made with <3 and ReactJS with Electron.
We are using the Storage and Database Service from Firebase.

### Set Up

* You should fill the `Firebase-clone` file with your own Firebase info. Once it´s done, you should change it´s name to `Firebase.js` in order for it to work correctly (Don´t worry, the file is being ignored by Git!), if you need my file, just ask me! It´s a demo app, but i don´t want my keys to be commited anywhere. 
* To start the app run `yarn electron-dev`
